# Boltx Ecto

Boltx Ecto is an adapter for Ecto using Boltx.


## Features

| Feature               | Implemented  |
| --------------------- | ------------ |
| Repo Adapter          | NO           |
| Schemas               | NO           |
| Queryable             | NO           |
| Transactions          | NO           |
| Migrations            | NO           |


## Installation

Add :boltx_ecto to your dependencies:

```elixir
def deps do
  [
    {:boltx_ecto, "~> 0.0.0"}
  ]
end
```
